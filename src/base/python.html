Introduction

Dans cette première section nous allons apprendre les briques élémentaires de la programmation. C'est à dire, qu'est ce que :

  • une variable
  • une condition
  • une boucle
  • une fonction
  • un type

Pour ce faire, nous utiliserons le langage Python. Qui est le langage avec la syntaxe la plus simple à utiliser.

Cette partie est destinée aux personnes voulant débuter en informatique. Si vous connaissez déjà les bases, vous pouvez directement passer à la partie suivante.



Installation

La première étape est donc de l'installer. Toutefois, si vous voulez un éditeur plus complet il est aussi possible d'installer Spyder. Enfin, si vous voulez les 2 ainsi que d'autres éditeurs utiles vous pouvez installer Winpython ou Anaconda, qui ne possède pas l'éditeur de base. Si vous débutez tout juste dans la programmation, je vous conseille l'éditeur de base qui est très simple d'utilisation. Vous pouvez toujours installer les autres plus tard.

Une fois installé, lancer idle (qui vient avec python), ou Spyder.

Si vous avez choisi idle, vous vous retrouvez sur un interpréteur, c'est-à-dire que tout ce que vous tapez est directement exécuté.
Si vous marquez 2+2 et tapez entrer, l'interpréteur vous répondra 4.
Afin de pouvoir écrire un programme (donc un ensemble de ligne) vous devez ouvrir un nouveau fichier (file -> new file ou ctrl+n). Une nouvelle fenêtre s'affiche, dans laquelle vous pouvez entrer autant de ligne de code que vous voulez.
Pour exécuter le code il faut alors appuyer sur run ou sur la touche f5.

Attention : il faut à présent rentrer du code. Donc on ne peut plus juste marquer 2+2.



Hello World !

Par tradition et afin de vous présenter une fonction utile, print, nous allons coder votre tout premier programme Hello World !
Cette fonction permet d'afficher soit un nombre print(2+2) soit une chaine de caractère qui doit donc se trouver "entre guillemet".
Ainsi, vous pouvez tapez et exécuter votre tout premier programme :

print("Hello World !)

Si tout se passe bien, le programme devrait avoir écrit "Hello World !".

Bien, maintenant que tout est en place, passons aux différentes notions.



Variable

Une variable est un espace de stockage, comme un carton vide labellé.
En python, on déclare et assigne les variables directement. Autrement dit, si je veux créer une variable qui s'appelle maVariable et qui a pour valeur 1, il me suffit d'écrire :
maVariable = 1
Je peux même l'afficher :
print(maVariable)

En suivant le même principe, je peux changer sa valeur : maVariable = 5
Attention, son ancienne valeur est à présent perdue et non retrouvable.

Jusqu'à présent nous avons stocker des entiers, mais il est possible de stocker n'importe quoi, comme une chaîne de caractère à présent :
uneAutreVariable = "du texte quelconque"

Note : Le nom d'une variable doit commencer par une lettre et ne peut pas contenir d'espace.
Par convention, la première lettre est minuscule et on utilise la majuscule pour "séparer" les mots, comme ci dessus.



Auto-Assignation

En informatique, il ne faut pas confondre == et = .
Le 1er sert de test. "Est-ce que 2 valeurs sont identique" (dans l'interpréteur idle vous pouvez tapez 2==3 et il vous répondra false)
Le 2ème sert d'assignation. "La valeur de gauche prend la valeur de droite". Il est important de comprendre que la valeur de gauche est celle en devenir et celle de droite la valeur au moment de la lecture de cette ligne et avant assignation.
En quoi est-ce important ? C'est très simple, en informatique il fréquent d'écrire :
maVariable = maVariable + 1
Ce qui mathématiquement n'a pas de sens, ou plutôt pas de solution. Mais, ici, ce que l'on veut dire c'est "maVariable prend pour valeur la valeur qu'elle avait plus un".
Ainsi, si maVariable avait pour valeur 5 elle vaut à présent 6. Vous pouvez tester avec print.



Condition

Une condition permet d'exécuter une partie de code sous la condition qu'un test soit vrai.
Autrement dit, si une condition est vraie on fait quelque chose, sinon autre chose. En python, cela s'écrit :


                    maVariable = 5
                    if( maVariable == 5 ):
                        maVariable = 3
                    else:
                        maVarible = 7

Ici, on compare la valeur de maVariable à l'entier 5. S'ils sont égaux on fait une assignation, sinon on fait une autre assignation.

Note : On remarque déjà l'importance de ne pas confondre les différents "égals". Il existe une 3ème commande en plus de if et else. Il s'agit de elif (en python), qui permet d'enchainer les if avant de finir avec le else qui sert de cas par défaut. 


                    maVariable = 5
                    if( maVariable == 5 ):
                        maVariable = 3
                    elif( maVariable == 3 ):
                        maVariable = 78
                    elif( maVariable == 6 ):
                        maVariable = 4
                    else:
                        maVarible = 7

Note : Le code se lit de haut en bas et dès lors qu'un des tests est vrai. Le programme ignore les suivant jusqu'à la fin du else. Et ne fait le else que si tous les autres tests ont échoué.
Dans l'exemple ci dessus, à la fin du code maVariable ne vaut pas 78 mais bien 3.
Enfin, il est important de noter les tabulations et les ":" qui sont important en python.



Boucle

Une boucle sert à répéter une partie de code un certain nombre de fois. Il existe 2 type de boucle :

  • la boucle for : qui permet de répéter le code un nombre fixe de fois
  • la boucle while : qui permet de répéter un code tant qu'une condition est vraie

En python, on les utilise de cette manière : 


                    maVariable = 0
                    for i in range(10):
                        maVariable = maVariable + 1
                    print(maVariable)
                    

                    maVariable = 5
                    while( maVariable < 10 ):
                        maVariable = maVariable + 1
                    print(maVariable)

Dans ces 2 cas simple la maVariable vaut 5 au final. Mais dans le 1er cas on a fait 10 iétrations alors que dans le 2ème on en a fait autant qu'il en fallait pour arriver à 10 (donc 5 ici).
