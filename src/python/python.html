Introduction

Dans cette première section nous allons apprendre les briques élémentaires de la programmation. C'est à dire, qu'est ce que :

  • une variable
  • un type
  • une condition
  • une boucle
  • une fonction

Pour ce faire, nous utiliserons le langage Python. Qui est le langage avec la syntaxe la plus simple à utiliser.

Cette partie est destinée aux personnes voulant débuter en informatique. Si vous connaissez déjà les bases, vous pouvez directement passer à la partie suivante.



Installation

La première étape est donc de l'installer. Toutefois, si vous voulez un éditeur plus complet il est aussi possible d'installer Spyder. Enfin, si vous voulez les 2 ainsi que d'autres éditeurs utiles vous pouvez installer Winpython ou Anaconda, qui ne possède pas l'éditeur de base. Si vous débutez tout juste dans la programmation, je vous conseille l'éditeur de base qui est très simple d'utilisation. Vous pouvez toujours installer les autres plus tard.

Une fois installé, lancer idle (qui vient avec python), ou Spyder.

Si vous avez choisi idle, vous vous retrouvez sur un interpréteur, c'est-à-dire que tout ce que vous tapez est directement exécuté.
Si vous marquez 2+2 et tapez entrer, l'interpréteur vous répondra 4.
Afin de pouvoir écrire un programme (donc un ensemble de ligne) vous devez ouvrir un nouveau fichier (file -> new file ou ctrl+n). Une nouvelle fenêtre s'affiche, dans laquelle vous pouvez entrer autant de ligne de code que vous voulez.
Pour exécuter le code il faut alors appuyer sur run ou sur la touche f5.

Attention : il faut à présent rentrer du code. Donc on ne peut plus juste marquer 2+2.



Hello World !

Par tradition et afin de vous présenter une fonction utile, print, nous allons coder votre tout premier programme Hello World !
Cette fonction permet d'afficher soit un nombre print(2+2) soit une chaine de caractère qui doit donc se trouver "entre guillemet".
Ainsi, vous pouvez tapez et exécuter votre tout premier programme :

print("Hello World !)

Si tout se passe bien, le programme devrait avoir écrit "Hello World !".

Bien, maintenant que tout est en place, passons aux différentes notions.

Quelques informations utiles :
Pour une ligne donnée, tout ce qui suit un # est considéré comme un commentaire (en python) : 


                        #Ceci est un commentaire
                        print("cela s'affiche")  # Ici aussi, print("cela ne s'affiche pas")
Enfin, si une exécution ne se termine pas (boucle infini) vous pouvez utiliser le raccourci ctrl+c dans l'interpréteur (là où les résultats s'affichent) pour arrêter le programme.



Variable

Une variable est un espace de stockage, comme un carton vide labellé.
En python, on déclare et assigne les variables directement. Autrement dit, si je veux créer une variable qui s'appelle maVariable et qui a pour valeur 1, il me suffit d'écrire :
maVariable = 1
Je peux même l'afficher :
print(maVariable)

En suivant le même principe, je peux changer sa valeur : maVariable = 5
Attention, son ancienne valeur est à présent perdue et non retrouvable.

Jusqu'à présent nous avons stocker des entiers, mais il est possible de stocker n'importe quoi, comme une chaîne de caractère à présent :
uneAutreVariable = "du texte quelconque"

Note : Le nom d'une variable doit commencer par une lettre et ne peut pas contenir d'espace.
Par convention, la première lettre est minuscule et on utilise la majuscule pour "séparer" les mots, comme ci dessus.



Auto-Assignation

En informatique, il ne faut pas confondre == et = .
Le 1er sert de test. "Est-ce que 2 valeurs sont identique" (dans l'interpréteur idle vous pouvez tapez 2==3 et il vous répondra false)
Le 2ème sert d'assignation. "La valeur de gauche prend la valeur de droite". Il est important de comprendre que la valeur de gauche est celle en devenir et celle de droite la valeur au moment de la lecture de cette ligne et avant assignation.
En quoi est-ce important ? C'est très simple, en informatique il fréquent d'écrire :
maVariable = maVariable + 1
Ce qui mathématiquement n'a pas de sens, ou plutôt pas de solution. Mais, ici, ce que l'on veut dire c'est "maVariable prend pour valeur la valeur qu'elle avait plus un".
Ainsi, si maVariable avait pour valeur 5 elle vaut à présent 6. Vous pouvez tester avec print.



Type

Simple

Un type représente la "catégorie" dans laquelle se trouve une variable. Les types de bases sont les suivants :

  • int (ou integer) : ce sont les entiers (ex : -5, 0, 7, 115, ...)
  • float : ce sont les nombres à virgule (ex : 1.1, 78.89541, 5.0, 7, ...). A noter qu'en informatique on utilise un point pour les virgules.
  • str (ou string) : ce sont les chaînes de caractères, représentés entre guillemets (ex : "Bonjour", "ceci est un string", "a", "83", ...)
  • char (ou characteres) : ce sont eux qui composent les strings, représentés entre apostrophes (ex : 'a', 's', ...)


Liste

Une liste contient plusieurs valeurs/variables de même type. Elle est représentée entre crochet, et les éléments sont séparés par des virgules :


listeDeInt = [1, 2, 78, 95, 5, -6, 75]
listeDeJour = ["lundi", "mardi", "mercredi", "jeudi", "vendredi", "samedi", "dimanche"]
On peut accéder à un élément d'une liste grâce à son index. Il est toutefois, important de noter qu'en informatique, les index vont de 0 à tailleDeLaListe-1
Ainsi, listeDeInt[3] vaut 95, listeDeInt[1] vaut 2 et listeDeJour[0] vaut "lundi".

Quelques astuces :
En python, on peut utiliser des indices négatifs afin de partir de la fin. Ainsi, listeDeJour[-1] vaut "dimanche".
On peut utiliser la fonction len pour avoir le nombre d'éléments dans une liste. Ainsi, listeDeJour vaut 7 et ses index vont de 0 à len(listeDeJour)-1 (donc 6, ici).



Condition

Une condition permet d'exécuter une partie de code sous la condition qu'un test soit vrai.
Autrement dit, si une condition est vraie on fait quelque chose, sinon autre chose. En python, cela s'écrit :


                    maVariable = 5
                    if( maVariable == 5 ):
                        maVariable = 3
                    else:
                        maVariable = 7

Ici, on compare la valeur de maVariable à l'entier 5. S'ils sont égaux on fait une assignation, sinon on fait une autre assignation.

Note : On remarque déjà l'importance de ne pas confondre les différents "égals". Il existe une 3ème commande en plus de if et else. Il s'agit de elif (en python), qui permet d'enchainer les if avant de finir avec le else qui sert de cas par défaut. 


                    maVariable = 5
                    if( maVariable == 5 ):
                        maVariable = 3
                    elif( maVariable == 3 ):
                        maVariable = 78
                    elif( maVariable == 6 ):
                        maVariable = 4
                        maVariable = maVariable + 1
                    else:
                        maVariable = 7
                    uneAutreVariable = 0

Note : Le code se lit de haut en bas et dès lors qu'un des tests est vrai. Le programme ignore les suivant jusqu'à la fin du else. Et ne fait le else que si tous les autres tests ont échoué.
Dans l'exemple ci dessus, à la fin du code maVariable ne vaut pas 78 mais bien 3.
Enfin, il est important de noter les tabulations et les ":" qui sont important en python. En effet, les tabulations servent à délimiter le code qui dépend de la condition. Ainsi, si maVariable==6 est vraie, alors les 2 lignes suivantes sont exécutées. De même, le else ne comprend qu'une seule ligne et la dernière ligne fait partie du code "normal" qui sera exécuté sans condition.



Boucle

Une boucle sert à répéter une partie de code un certain nombre de fois. Il existe 2 type de boucle :

  • la boucle for : qui permet de répéter le code un nombre fixe de fois
  • la boucle while : qui permet de répéter un code tant qu'une condition est vraie

En python, on les utilise de cette manière : 


                    maVariable = 0
                    for i in range(10):
                        maVariable = maVariable + 1
                    print(maVariable)
                    

                    maVariable = 5
                    while( maVariable < 10 ):
                        maVariable = maVariable + 1
                    print(maVariable)

Dans ces 2 cas simples la maVariable vaut 10 au final. Mais dans le 1er cas on a fait 10 itérations alors que dans le 2ème on en a fait autant qu'il en fallait pour arriver à 10 (donc 5 ici).

Il y a plusieurs éléments important à noter :
De la même façon que pour les conditions, les tabulations et les ":" sont importants et jouent le même rôle que précédemment.

Pour la boucle for :
for i in range(10) peut se traduire par "On crée une variable i qui va prendre, à tour de rôle, la valeur de 0 à 9".
Quelques points importants : la variable i n'existe qu'au sein de la boucle for et n'est plus accessible en dehors. Une méthode simple pour savoir si une variable existe ou pas et de regarder les tabulations. Si une variable est créée à un niveau de tabulations n alors elle est accessible à tous les niveaux supérieurs ou égaux à n et inaccessible à tous les niveaux inférieurs à n. De plus, comme remarqué précédemment, en informatique les index commencent à 0 !
Donc range(n) permet d'accéder aux nombres de 0 à n-1. Et range(n, m) permet d'accéder au nombre de n à m-1. Enfin, la variable i prend la prmeière valeur au début du 1er tour. Et change de valeur à chaque fin de tour jusqu'à ce qu'il n'y en ait plus de disponible.


Pour la boucle while :
Le fonctionnement est le suivant :

  1. On teste la condition
  2. Si elle est vraie on exécute le code et on reteste la condition
  3. On continue jusqu'à ce que la condition soit fausse
Ceci implique que l'on peut faire 0 itérations, si la condition est fausse instantanément. Mais surtout, on peut rentrer dans une boucle infinie !
Ceci arrive lorsque la condition, au contraire, n'est jamais fausse. Il est très important de vérifier de ne pas en créer accidentellement.
Ici, on attend que maVariable soit supérieure ou égale à 10. Et l'on augmente sa valeur à chaque itération. Donc la condition finira par être vraie.
Rappel : Pour arrêter une boucle infinie, avant qu'elle fasse planter votre ordinateur, appuyer sur ctrl+c dans l'interpréteur (là où le code s'exécute).


Imbrication

Enfin, il est important de noter que l'on peut avoir plusieurs niveaux de tabulations qu'il faut donc aligner correctement, afin d'avoir l'effet voulu. 


                        maVariable = 0
                        variable2 = 0
                        if(variable2 > maVariable):
                          maVariable = variable2
                          for i in range(10):
                            variable2 = variable2 + 2
                            if(variable2 > maVariable + 10):
                              maVariable = 100
                        else:
                          maVariable = -100
Ici, il y a 3 niveaux d'imbrication et l'alignement des tabulations permet de savoir le comportement.
Par exemple, nous pouvons voir que le else final se joue si le 1er if est faux alors que le 2ème if ne possède pas de else.



Fonction

Une fonction est une partie de code que l'on peut rappeler. Ainsi, si votre code permet de trouver le nombre minimum d'une liste et que vous avez 2 listes, il n'est pas la peine de tout retaper. En python, une fonction se définit ainsi : 


                        def maFonction():
                          print("Ceci est une fonction")
                        maFonction()
                        maFonction()
Ici, on définit une fonction qui écrit un texte, et on l'appelle 2 fois.

Il est possible de donner des arguments à la fonction. Ce sont comme des variables neutres. elles ne valent rien de spécial lors de la définition et prennent une valeur lors de l'exécution : 


                        def maFonctionAddition(argument1, argument2):
                          print(argument1 + argument2)
                        maFonctionAddition(5, 15)
Ici, notre fonction prend 2 arguments et écrit leur somme.

Enfin, à la place, ou en plus, décrire un résultat, une fonction peut le retourner. C'est-à-dire, permettre de le récupérer dans la suite du programme pour l'utiliser. 


                        def additionneur(x, y):
                          resultat = x + y
                          return resultat

                        test = additionneur(11, 78)
                        print(test)
Ici, à la place d'afficher le résultat de l'addition, le fonction renvoie ce résultat et on le récupère dans une variable que l'on affiche, et que l'on pourrait utiliser pour d'autre chose.

Import

Comme nous l'avons vu, il existe un certain nombre de fonctions pré-défini en python, tel que print() ou len(). Toutefois, afin de ne pas charger trop de fonctions inutiles pour un code donné, elles ne sont pas toutes accessibles directement, et certaines doivent être importées.
Les imports se définissent tout en haut de la page de code, traditionnellement. 


        from numpy import array #importe la fonction array du module numpy
        from numpy import * #importe tout le module numpy
        import numpy as np #importe le module numpy et le renomme np

Comme nous venons de le voir, en python, il existe plusieurs façons d'importer un module ou une fonction. Le renommage est utile, puisque pour utiliser une fonction d'un module on doit taper : nomDuModule.nomDeLaFonction().
Ici, numpy.array([1, 2, 3, 4]) ou np.array([1, 2, 3, 4]).

Note : Si votre interpréteur vous informe qu'il ne connaît pas "numpy", il est temps de changer d'interpréteur et d'utiliser spyder.
Il est possible d'avoir les imports en utilisant python de base, sans doute via un autre téléchargement.
